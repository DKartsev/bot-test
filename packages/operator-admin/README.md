# operator-admin

Админ-панель операторов на Next.js.

## Установка

```bash
npm install
```

## Запуск разработки

```bash
npm run dev
```

## Переменные окружения

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

Создайте `.env` файл и задайте значения для этих переменных перед запуском.
